ABAP Classic Outline - Changelog version 1.0.2

Front-end changes

  • Help content added to plugin
  • Preferences window added in ABAP Development Category
  • Fetch redefinitions for methods parameter
    If latest backend component is installed and this parameter is enabled, the redefinitions of methods in subclasses are displayed under the method node.
  • Load all levels of sublcass parameter
    If latest backend component is installled and this parameter is enabled, then in the subclasses node, all subclasses from all levels will be visible. When disabled, then only first level of inheritance will be shown.
    In case of previous backend version, parameter has no effect and all levels will be visible.
  • Icon decoration added to display static, read-only, abstract, constructors, etc.
    Needs latest back-end component
  • Item counter added
